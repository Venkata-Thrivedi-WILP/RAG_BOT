**Overview**
------------

Delete authentication profile endpoint allows users to remove their existing authentication profiles. This endpoint is used when a user wants to delete their stored credentials.

To use this endpoint, you need to send a POST request to `/AuthProfile/DeleteProfile` and include the `uuid` of the authentication profile you want to delete in the request body.

**Security Note:** This endpoint requires bearer authentication.

**Developer Notes**
------------------

### Parameters

* `uuid` (query, Required): Authentication UUID.

### Request Body

* `Content Type`: application/json
* `Schema Properties`:
	+ `uuid` (string): The authentication profile uuid either passed in by method call or as part of the payload.
	+ `Required fields`: `uuid`

### Response

* Status Code: 200
* Description: API-Result
* Content Type: `*/*`
* Response Schema: AuthProfileDeleteProfile
* Response Body Properties:
	+ `Result` (boolean): Success or failure of the delete
	+ `Error` (object): Error message text on failure, may be null

### Sample Request JSON
```json
{
  "uuid": "string_value"
}
```

### Sample Response JSON
```json
{
  "Result": false,
  "Error": {}
}
```
