# Overview  
The **Get Authentication Profile** endpoint allows users to retrieve an authentication profile using a unique identifier (UUID). This is done via a POST request to the path `/AuthProfile/GetProfile`. The endpoint requires a valid bearer token for authentication, ensuring that only authorized users can access the profile information.

# Developer Notes  
- **Required Parameters:**  
  - `uuid` (query, Required): The UUID of the profile to be retrieved.  

- **Request Structure:**  
  - Content Type: `application/json`  
  - Required fields:  
    - `uuid` (string): The authentication profile UUID.  

- **Sample Request JSON:**  
  ```json
  {
    "uuid": "string_value"
  }
  ```

- **Response Structure:**  
  - Status Code: `200`  
  - Content Type: `*/*`  
  - Response Schema: `AuthProfileGetProfile`  

- **Error Handling:**  
  - The response may include an `Error` object containing error message text on failure, which may be null.  

- **Security Considerations:**  
  - This endpoint requires bearer token authentication.  

# Detailed Explanation of Available Data, Request and Response Parameters  
## Request Parameters  
- **uuid**  
  - **Type:** string  
  - **Description:** The authentication profile UUID either passed in by method call or as part of the payload.  
  - **Example Value:** `"123e4567-e89b-12d3-a456-426614174000"`  

## Response Parameters  
- **Result** (object)  
  - **Description:** Contains the authentication profile details.  
  - **Properties:**  
    - **Challenges** (array): Selected authentication mechanisms.  
      - **Array Items:** string values representing different challenges.  
    - **AdditionalData** (object): DataEntity with additional data (default=null).  
    - **Uuid** (string): Profile UUID.  
    - **SingleChallengeMechanisms** (string): Selected single authentication mechanisms.  
    - **DurationInMinutes** (integer): How many minutes the profile is valid.  
    - **Name** (string): Profile name.  
- **Error** (object): Error message text on failure, may be null.  

## Sample Response JSON:  
```json
{
  "Result": {
    "Challenges": [
      "string_value"
    ],
    "AdditionalData": {},
    "Uuid": "string_value",
    "SingleChallengeMechanisms": "string_value",
    "DurationInMinutes": 0,
    "Name": "string_value"
  },
  "Error": {}
}
```
