**Overview**
-----------

Create a tag with no apps for the current user. This endpoint allows users to create a new tag without assigning it to any app.

* HTTP Method: POST
* Path: `/UPRest/CreateTagWithNoApp`
* Security Requirement: Bearer Authentication
