# Overview
The **Get an application's data** endpoint allows users to retrieve detailed information about a specific application using its unique application key. This endpoint utilizes the **POST** HTTP method and is accessed via the path **/UPRest/GetAppByKey**. To use this endpoint, authentication is required through a bearer token.

# Developer Notes
- **Required Parameters:**
  - `appkey` (query, Required): The application key used to identify the specific application.
  
- **Request Structure:**
  - Content Type: `application/json`
  - Request Body: Must include the `markAppVisited` property.
  
- **Response Structure:**
  - Status Code: `200` for successful requests.
  - Response Body includes:
    - `Result`: Contains application data.
    - `Error`: Contains error message text on failure (may be null).
  
- **Security Considerations:**
  - Requires bearer authentication for access.

# Detailed Explanation of Available Data, Request and Response Parameters
### Request Parameters
- **appkey** (query, Required)
  - Type: string
  - Description: Unique key for the application.
  - Example: `appkey=12345`

### Request Body
- **markAppVisited** (string)
  - Description: Indicates whether the application should be marked as visited.
  - Example: 
  ```json
  {
    "markAppVisited": "true"
  }
  ```

### Response Parameters
- **Result** (object)
  - Contains detailed information about the application:
    - **Shortcut** (boolean): Indicates if the application is set for optional installation.
    - **Url** (string): URL of the application.
    - **AdminTag** (string): Admin tag for the application.
    - **Description** (string): Description of the application.
    - **Category** (string): Category of the application.
    - **Name** (string): Name of the application.
    - **TemplateName** (string): Template name of the application.
    - **PasswordIsSet** (boolean): Indicates if a password is set for the application.
    - **WebAppType** (string): Type of the web application.
    - **DisplayName** (string): Display name of the application.
    - **UsernameRO** (boolean): Indicates if the username is read-only.
    - **WebAppTypeDisplayName** (string): Display name of the web app type.
    - **AppTypeDisplayName** (string): Display name of the application type.
    - **Intranet** (boolean): Indicates if the application is an intranet app.
    - **AppType** (string): Type of the application.
    - **AppKey** (string): Application key.
    - **Rank** (integer): Rank of the application.
    - **Icon** (string): Path to the application icon image.
    - **Username** (string): Username for the application.
  
- **Error** (object)
  - Contains error message text on failure, may be null.
