**Overview**
================

Returns a list of credential providers for import. This endpoint is used to retrieve a list of available credential providers that can be imported into the system.

* HTTP Method: POST
* Path: `/UPRest/GetCredentialsProviderListForImport`
* Security Requirement: Bearer Authentication
