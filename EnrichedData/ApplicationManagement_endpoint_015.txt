# Overview  
This endpoint allows users to retrieve a list of credential providers for import purposes. It uses the HTTP POST method and is accessed via the path `/UPRest/GetCredentialsProviderListForImport`. To use this endpoint, a bearer authentication token is required to ensure secure access.
