**Overview**
-----------

The `/UPRest/GetRecentImportedAccountsFile` endpoint allows users to retrieve a list of details for recently uploaded files. This endpoint is part of the Application Management tags and requires authentication using bearer tokens.

To use this endpoint, you will need to send a POST request to `https://api.example.com/UPRest/GetRecentImportedAccountsFile`. You can specify the number of recent files you want to retrieve by including the `fileCount` query parameter in your request.

**Security Note:** This endpoint requires bearer authentication. Please ensure that your API token is properly formatted and included in the `Authorization` header of your request.
