# Overview
This API endpoint allows you to retrieve the applications that a specific user has access to. It is designed for use by system administrators and users with application management or user management rights. The endpoint uses the HTTP POST method and is located at `/UPRest/GetResultantAppsForUser`. Access to this API is secured and requires bearer authentication. Note that personal applications are not included in the results, and the list of applications and roles is cached.

# Developer Notes
- **Required Parameters:**
  - `userUuid` (query, Required): The unique ID of the user whose applications are being retrieved. This ID is generated when the user is created or invited.
  
- **Request Structure:**
  - HTTP Method: POST
  - Path: `/UPRest/GetResultantAppsForUser`
  
- **Response Structure:**
  - Status Code: 200 for successful requests.
  - Content Type: `*/*`
  - Response Schema: `UPRestGetResultantAppsForUser`
  
- **Error Handling:**
  - If the user ID is invalid or the user lacks the necessary rights, an error message will be returned indicating authorization failure.

- **Security Considerations:**
  - This API requires bearer authentication.

# Detailed Explanation of Available Data, Request and Response Parameters

### Request Parameters
- **userUuid** (string, Required): 
  - Description: The unique ID of the user.
  - Example: `"cb9b5761-6cfe-45a5-8ecf-ce9fa9e0ff82"`

### Response Parameters
- **Result** (object): 
  - Contains the result of a successful API call.
  - **ErrorID** (string): Error ID if any error is received. Default is 'null'.
  - **ErrorCode** (string): Error code if any error is received. Default is 'null'.
  - **Exception** (string): Exception message if any exception is received. Default is 'null'.
  - **MessageID** (string): Message ID for failure response. Default is 'null'.
  - **Result** (object): 
    - **ReturnID** (string): Return ID for the successful API call. Default is 'null'.
    - **Count** (integer): Number of records returned.
    - **Columns** (array): Array of column definitions.
      - **TableKey** (string): Specifies key type (e.g., 'Primary').
      - **Description** (string): Brief description of the column.
      - **Name** (string): Name of the column.
      - **IsHidden** (boolean): Indicates if the column is hidden. Default is false.
      - **Type** (string): Numeric indicator of the data type.
      - **Title** (string): Title of the column.
    - **Results** (object): 
      - **Shortcut** (boolean): Indicates if the application is set for optional installation. Default is false.
      - **Entities** (array): List of entities.
        - **IsForeignKey** (string): Indicates if the entity has a foreign key. Default is false.
        - **Type** (string): Type of the entity (e.g., Application).
        - **Key** (string): Unique ID of the Entity.
      - **Url** (string): Actual URL of external applications.
      - **BypassLoginMFA** (boolean): Indicates if MFA can be bypassed. Default is false.
      - **Row** (string): Details about user-accessible applications.
      - **Category** (string): Category of the application (e.g., Finance, CRM).
      - **Name** (string): Name of the application.
      - **DisplayName** (string): Display name of the application.
      - **AppKey** (string): Unique ID of the application.
      - **IsNewApp** (boolean): Indicates if the application is newly added. Default is false.
      - **Icon** (string): Absolute path of the application's icon image.
      - **FullCount** (integer): Number of tables.
      - **success** (boolean): Indicates success of the API call.
  - **InnerExceptions** (string): Details of inner exceptions.
  - **Message** (string): Error message for failure response. Default is 'null'.
  - **Error** (object): Error message text on failure.
    - **UnAuthorized** (string): Message displayed for unauthorized access.
