**Overview**
-----------

Update secured item credentials endpoint allows users to update the password, custom fields, notes, and username for a secured item. This endpoint is used in application management and requires authentication using the Bearer token.

* HTTP Method: POST
* Path: `/UPRest/UpdateCredsForSecuredItem`
* Security Requirements: bearerAuth

**Developer Notes**
------------------

### Required Fields

* `Password` (string): User password
* `CustomFields` (string): Custom user fields
* `Notes` (string): Notes for secured item
* `Username` (string): Username

### Request/Response Structure

* Request Body: application/json, schema properties as described above
* Response Body:
	+ `Result` (object): Whether the operation is successful or not.
	+ `Error` (object): Error message text on failure, may be null.
