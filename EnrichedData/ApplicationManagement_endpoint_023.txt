**Overview**
===============

Update personal user application endpoint allows users to update their application details. This endpoint is used for managing applications and requires authentication via bearer token.

* HTTP Method: POST
* Path: `/UPRest/UpdatePersonalApplication`
* Security Requirement: Bearer Authentication

### Required Fields

### Request/Response Structure

* Request Body: JSON object with required fields (`appname`, `appkey`, `appDescription`, `notes`, and `appUrl`)
* Response Body:
	+ `Result` (object): Whether the operation is successful or not
	+ `Error` (object): Error message text on failure, may be null

### Security Considerations

* Bearer Authentication required for all requests
