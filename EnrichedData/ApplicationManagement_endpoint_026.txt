**Overview**
-----------

The Upload secured item icon endpoint allows users to upload a secured item icon. This endpoint is used in Application Management and requires authentication using the bearer token.

To use this endpoint, you need to send a POST request to `/UPRest/UploadSecuredItemIcon` with a JSON payload containing the `sItemkey` field. The request body must be in the format specified by the schema properties.
