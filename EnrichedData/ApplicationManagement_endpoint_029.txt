# Overview  
This endpoint retrieves all available data for the user portal in a single call. It uses the POST HTTP method and can be accessed at the path `/UPRest/GetUPData`. The call caches the list of applications but does not cache tag information. To access this endpoint, a bearer authentication token is required.

# Developer Notes  
- **Required Parameters:**  
  - `force` (query, Required): Whether to use the cache. If set to true, the call does not use the cache.  
  - `username` (query, Required): Target user name.  

- **Request Body:**  
  - Content Type: `application/json`  
  - Schema Properties: Required  

- **Response Structure:**  
  - Status Code: `200`  
  - Content Type: `*/*`  
  - Response Schema: `UPRestGetUPData`  

- **Error Handling:**  
  - The response may contain an `Error` object with an error message text on failure, which may be null.  

- **Security Considerations:**  
  - Bearer authentication is required to access this endpoint.  

# Detailed Explanation of Available Data, Request and Response Parameters  
## Request Parameters  
- **force** (query, Required)  
  - Type: Boolean  
  - Description: Determines whether to use the cache. Set to `true` to bypass the cache.  
  - Example: `?force=true`  

- **username** (query, Required)  
  - Type: String  
  - Description: The target user's name for whom the application data is being retrieved.  
  - Example: `?username=johndoe`  

## Response Parameters  
- **Result** (object): A list of applications with the following nested properties:  
  - **Shortcut** (boolean): Indicates if the application has been set for optional installation.  
  - **_RowKey** (string): Application ID.  
  - **Url** (string): URL of the application.  
  - **Description** (string): Description of the application.  
  - **Category** (string): Category of the application.  
  - **AdminTag** (string): Admin tag for the application.  
  - **Name** (string): Name of the application.  
  - **TemplateName** (string): Template name of the application.  
  - **WebAppType** (string): Type of the web application.  
  - **PasswordIsSet** (boolean): Indicates if the password has been set.  
  - **DisplayName** (string): Display name of the application.  
  - **UsernameRO** (boolean): Indicates if the username is read-only.  
  - **WebAppTypeDisplayName** (string): Display name of the web application type.  
  - **Personal** (boolean): Indicates if the application is a self-service application.  
  - **AppTypeDisplayName** (string): Display name of the application type.  
  - **AppType** (string): Type of the application.  
  - **Intranet** (boolean): Indicates if the application is an intranet app.  
  - **AppKey** (string): Key for the application.  
  - **Automatic** (boolean): Indicates if the application has been set for automatic installation.  
  - **Rank** (integer): Rank of the application.  
  - **Icon** (string): Path of the application icon image.  
  - **Username** (string): Username for the application.  

- **Error** (object): Contains error message text on failure, may be null.
