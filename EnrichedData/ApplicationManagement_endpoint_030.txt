**Overview**
==========

Verify if a username provided for a personal app or secured item is allowed. This endpoint accepts a POST request to `/UPRest/ValidateUsernameIsAllowed`. The request body must contain a JSON payload with a single field `userName` of type string.

Security: Bearer authentication is required for this endpoint.

**Developer Notes**
==================

### Request Parameters

* `userName` (string): WPM user name
	+ Required: Yes
	+ Example: "string_value"

### Response Structure

* `Result` (object): Whether the operation is successful or not.
	+ Example: `{}` (empty object)
* `Error` (object): Error message text on failure, may be null.
	+ Example: `{}` (empty object)

### Request/Response Body

* Request body:
	+ Content type: application/json
	+ Schema properties: `userName` (string) is required
	+ Sample request JSON:
```json
{
  "userName": "string_value"
}
```
* Response body:
	+ Content type: */*
	+ Response schema: UPRestCreateTagWithNoApp

### Security Considerations

* Bearer authentication is required for this endpoint.
