**Overview**
================

This endpoint allows you to set the TOTP seed for a secured password. The HTTP POST request is used, and the path is `/UPRest/SaveTotpSeedForSecuredPassword`. This API is part of the Application Management tags.

To use this endpoint, you need to provide a JSON body with the required parameters: `sItemkey`, `algorithm`, `digits`, `issuer`, `label`, `totpSecret`, and optionally `type` and `period`.

**Developer Notes**
================

### Required Fields

* `sItemkey`: The Secured Item's Unique ID
* `algorithm`: The algorithm used for TOTP verification code generation (default: SHA1)
* `digits`: This defines the number of verification code digits
* `issuer`: Issuer of the TOTP authentication key
* `label`: Label for the TOTP authentication key
* `totpSecret`: This is the TOTP authentication key secret for an application
* `type` (optional): Specifies the type of OTP being used
* `period` (optional): Verification code validity period (default: 30 seconds)
