# Overview
This API endpoint allows you to set the TOTP (Time-based One-Time Password) seed for a secured password. It is accessible via a POST request to the path `/UPRest/SaveTotpSeedForSecuredPassword`. If the `totpSecret` value is not provided, the system will clear the existing secret for the secured password. This endpoint requires bearer token authentication for security.

# Developer Notes
- **Required Parameters**: 
  - `sItemkey` (string): The Secured Item's Unique ID.
  - `totpSecret` (string): The TOTP authentication key secret for an application.
  
- **Request Structure**:
  - Content Type: `application/json`
  - Example Request JSON:
    ```json
    {
      "sItemkey": "string_value",
      "algorithm": "string_value",
      "digits": 0,
      "issuer": "string_value",
      "label": "string_value",
      "totpSecret": "string_value",
      "type": "string_value",
      "period": 0
    }
    ```

- **Response Structure**:

  - Response Content Type: `*/*`
  - Example Response JSON:
    ```json

- **Security Considerations**:

# Detailed Explanation of Available Data, Request and Response Parameters

## Request Parameters
- **sItemkey** (string): 
  - Description: The Secured Item's Unique ID.
  - Example: `"item123"`
  
- **algorithm** (string): 
  - Description: The algorithm used for TOTP verification code generation. Default is `SHA1`.
  - Example: `"SHA256"`
  
- **digits** (number): 
  - Description: Number of verification code digits.
  - Example: `6`
  
- **issuer** (string): 
  - Description: Issuer of the TOTP authentication key.
  - Example: `"MyApp"`
  
- **label** (string): 
  - Description: Label for the TOTP authentication key.
  - Example: `"User's TOTP"`
  
- **totpSecret** (string): 
  - Description: The TOTP authentication key secret for an application.
  - Example: `"JBSWY3DPEHPK3PXP"`
  
- **type** (string, optional): 
  - Description: Specifies the type of OTP being used.
  - Example: `"TOTP"`
  
- **period** (number): 
  - Description: Verification code validity period. Default is `30` seconds.
  - Example: `30`

## Response Parameters
- **Result** (object): 
  - Description: Return object for a successful API call.
  - Nested properties:
    - **success** (boolean): Indicates if the operation was successful.
    - **Result** (object): Result for a response. By default, it is `null`.
    - **Message** (string): Error message for failure response. By default, it is `null`.
    - **MessageID** (string): Message Id for failure response. By default, it is `null`.
    - **IsSoftError** (boolean): Indicates if this is a soft error (UI warning).
    - **Exception** (string): Exception message if any exception is received from the API. By default, it is `null`.
    - **ErrorID** (string): Error ID if any error is received from the API. By default, it is `null`.
    - **ErrorCode** (string): Error code if any error is received from the API. By default, it is `null`.
    - **InnerExceptions** (string): Inner exceptions.

- **Error** (object): 
  - Description: Error message text on failure.
  - Nested properties:
    - **UnAuthorized** (string): Error message when user does not have rights.
