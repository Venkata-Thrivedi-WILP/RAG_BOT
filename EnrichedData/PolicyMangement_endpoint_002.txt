# Overview  
The **Get Authentication Profile** endpoint allows users to retrieve an authentication profile using a unique identifier (UUID). This is accomplished through a POST request to the path `/AuthProfile/GetProfile`. To access this endpoint, users must provide a valid bearer token for authentication.

# Developer Notes  
- **Required Parameters:**  
  - `uuid` (query): The UUID of the authentication profile to be retrieved.  

- **Request Structure:**  
  - **Content Type:** application/json  
  - **Request Body:** Must include the `uuid`.  
  - **Sample Request JSON:**  
    ```json
    {
      "uuid": "string_value"
    }
    ```

- **Response Structure:**  
  - **Status Code:** 200  
  - **Response Body Properties:**  
    - `Result` (object): Contains the authentication profile details.  
    - `Error` (object): Contains error message text on failure, may be null.  

- **Security Considerations:**  
  - This endpoint requires bearer token authentication.

# Detailed Explanation of Available Data, Request and Response Parameters  

## Request Parameters  
- **uuid**  
  - **Type:** string  
  - **Description:** The authentication profile UUID, either passed in by method call or as part of the payload.  
  - **Constraints:** Required field.  
  - **Example:** `"uuid": "123e4567-e89b-12d3-a456-426614174000"`  

## Response Parameters  
- **Result** (object)  
  - **Description:** Contains the details of the authentication profile.  
  - **Properties:**  
    - **Challenges** (array): Selected authentication mechanisms.  
      - **Array items:** Strings representing different challenges.  
    - **AdditionalData** (object): DataEntity with additional data (default=null).  
    - **Uuid** (string): Profile UUID.  
    - **SingleChallengeMechanisms** (string): Selected single authentication mechanisms.  
    - **DurationInMinutes** (integer): How many minutes the profile is valid.  
    - **Name** (string): Profile name.  

- **Error** (object)  
  - **Description:** Contains error message text on failure, may be null.  

## Sample Response JSON  
```json
{
  "Result": {
    "Challenges": [
      "string_value"
    ],
    "AdditionalData": {},
    "Uuid": "string_value",
    "SingleChallengeMechanisms": "string_value",
    "DurationInMinutes": 0,
    "Name": "string_value"
  },
  "Error": {}
}
```
