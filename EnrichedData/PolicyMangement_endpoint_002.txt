**Overview**
-----------

The `Get Authentication Profile` endpoint allows users to retrieve their authentication profile. This endpoint is accessed via a POST request to the `/AuthProfile/GetProfile` path.

To use this endpoint, you will need to provide a valid `uuid` query parameter or include it in the request body. The response will be in JSON format and will contain an `Error` object if there was an issue with the request.

**Developer Notes**
------------------

### Required Parameters

* `uuid`: The authentication profile uuid either passed in by method call or as part of the payload.

### Request Response Structure

The response will be in JSON format and will contain an `Error` object if there was an issue with the request. The structure is as follows:

```json
{
  "Result": {
    "Challenges": [
      "string_value"
    ],
    "AdditionalData": {},
    "Uuid": "string_value",
    "SingleChallengeMechanisms": "string_value",
    "DurationInMinutes": 0,
    "Name": "string_value"
  },
  "Error": {}
}
```

### Security Considerations

This endpoint uses bearer authentication. Please ensure that you are passing a valid token in the `Authorization` header.
