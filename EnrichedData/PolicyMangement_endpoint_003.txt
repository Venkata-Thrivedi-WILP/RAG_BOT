# Overview
The **Set Authentication Profile** endpoint allows users to create or update an authentication profile. This is done by sending a POST request to the path `/AuthProfile/SaveProfile`. If the profile contains a UUID, the endpoint performs an update; otherwise, it creates a new profile. This endpoint requires bearer token authentication for security.

# Developer Notes
- **HTTP Method**: POST
- **Path**: `/AuthProfile/SaveProfile`
- **Security**: Requires bearer token authentication.
- **Required Parameters**:
  - `settings` (object): The authentication profile details.
- **Request Body**: Must be in JSON format.
- **Response Structure**:
  - On success (200): Returns an object containing the authentication profile.
  - On failure: Returns an error message.
- **Error Handling**: Check the `Error` field in the response for any failure messages.

# Detailed Explanation of Available Data, Request and Response Parameters
### Request Parameters
- **settings** (object): The authentication profile.
  - **Challenges** (array): Selected authentication mechanisms.
    - *Example*: `["multi-factor", "password"]`
  - **AdditionalData** (object): DataEntity with additional data (default=null).
    - *Example*: `{}`
  - **Uuid** (string): Profile UUID.
    - *Example*: `"123e4567-e89b-12d3-a456-426614174000"`
  - **SingleChallengeMechanisms** (string): Selected single authentication mechanisms.
    - *Example*: `"password"`
  - **DurationInMinutes** (integer): How many minutes the profile is valid.
    - *Example*: `60`
  - **Name** (string): Profile name.
    - *Example*: `"UserProfile"`

### Response Parameters
- **Result** (object): An authentication profile.
  - **Challenges** (array): Selected authentication mechanisms.
  - **AdditionalData** (object): DataEntity with additional data (default=null).
  - **Uuid** (string): Profile UUID.
  - **SingleChallengeMechanisms** (string): Selected single authentication mechanisms.
  - **DurationInMinutes** (integer): How many minutes the profile is valid.
  - **Name** (string): Profile name.
- **Error** (object): Error message text on failure, may be null.

### Sample Request JSON
