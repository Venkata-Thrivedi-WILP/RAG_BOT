**Overview**
================

Get authentication assurance level for a user's profile.

*   **Endpoint:** `/AuthProfile/GetProfileMFAScoring`
*   **HTTP Method:** `POST`
*   This endpoint retrieves the authentication assurance level (AAL) of a user's profile using Multi-Factor Authentication (MFA) scoring.
*   To use this endpoint, you must provide the required `challenges` query parameter.

**Developer Notes**
==================

### Required Parameters:

*   `challenges` (query, Required): Profile factors

### Request/Response Structure:

*   The request body should contain a JSON object with the required `challenges` parameter.
*   The response will be in JSON format and include a `Result` property containing the AAL values or an error message.

### Error Handling:

*   If there is an error with the request, the response will include an `Error` property with an error message text.
*   You can handle errors by checking for the presence of the `Error` property in the response.

### Security Considerations:

*   This endpoint uses bearer authentication. You must provide a valid token in the `Authorization` header to access this endpoint.
