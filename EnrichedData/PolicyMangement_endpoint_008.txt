# Overview
The **Get Policy Block** endpoint allows users to retrieve a specific policy block by its name. This is useful for managing and inspecting policy configurations within a system. The endpoint uses the HTTP POST method and is accessed at the path `/Policy/GetPolicyBlock`. To use this endpoint, authentication is required via a bearer token.

# Developer Notes
- **Required Parameters:**
  - `name` (query, Required): The name of the policy block to retrieve.
  
- **Request Structure:**
  - The request must be a POST request to the `/Policy/GetPolicyBlock` endpoint with the required query parameter.

- **Response Structure:**
  - On success, the response will return a status code of 200 with a body containing the policy block details.
  - The response will include a `Result` object and an `Error` object.

- **Error Handling:**
  - The `Error` object will provide details if the request fails.

- **Security Considerations:**
  - This endpoint requires bearer token authentication.

# Detailed Explanation of Available Data, Request and Response Parameters
## Request Parameters
- **name** (string, Required): 
  - Description: The name of the policy block to get.
  - Example: `"examplePolicyBlock"`

## Response Parameters
- **Result** (object): 
  - Description: The policy block details.
  - Nested properties:
    - **RiskAnalysisLevels** (object): 
      - Description: The risk analysis mapping.
    - **AuthProfiles** (array): 
      - Description: A list of authentication profiles.
      - Example: `[{}]`
    - **PolicyModifiers** (array): 
      - Description: The policy modifiers.
      - Example: `["string_value"]`
    - **RevStamp** (string): 
      - Description: The policy revision stamp.
      - Example: `"string_value"`
    - **RadiusClientList** (array): 
      - Description: The list of radius clients.
      - Example: `[{}]`
- **Error** (object): 
  - Description: Contains error details if the request fails.
  - Properties:
    - **description** (string): Description of the error.
    - **type** (string): Type of the error.
    - **properties** (object): 
      - Contains detailed properties similar to the Result structure.
