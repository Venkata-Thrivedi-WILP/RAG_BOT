**Overview**
-----------

Send Invitation Email Endpoint
================================

This endpoint allows system administrators to send login invitations to users. You must be a system administrator to use this endpoint.

* HTTP Method: POST
* Path: `/UserMgmt/SendLoginEmails`
* Security Requirement: Bearer Authentication

**Developer Notes**
------------------

### Required Parameters

* `ID`: List of user UUIDs to send invitations to (Array items: string)

### Request/Response Structure

* Request Body:
	+ Content Type: application/json
	+ Schema Properties:
		- `ID` (array): List of user UUIDs to send invitations to
			- Array items: string
			- Required fields: ID
* Response Body:
	+ `Result`: Standard result object
	+ `Error`: Error message text on failure, may be null

### Security Considerations

* Bearer Authentication required for system administrators only
