# Overview
The **Get User Details by Name** endpoint allows system administrators or users with user management permissions to retrieve detailed information about a specified cloud user. This endpoint uses the **POST** method and is accessed at the path **/CDirectoryService/GetUserByName**. To use this endpoint, the requester must have appropriate security credentials, specifically a bearer token for authentication.

# Developer Notes
- **Required Parameters:**
  - Request Body: Must include the `username` field.
  
- **Request Structure:**
  - Content Type: `application/json`
  - Example Request JSON:
    ```json
    {
      "username": "string_value"
    }
    ```

- **Response Structure:**
  - Status Code: `200` for successful requests.
  - Response Content Type: `*/*`
  - Response Schema: `CDirectoryServiceGetUserByName`
  
- **Error Handling:**
  - The response may include an `Error` object containing error message text on failure.

- **Security Considerations:**
  - Requires bearer token authentication.

# Detailed Explanation of Available Data, Request and Response Parameters
### Request Parameters
- **username (string)**: 
  - Description: The user name of the cloud user to fetch. Defaults to the current user if not specified.
  - Example: `"username": "john.doe"`

### Response Parameters
- **Result (object)**: Contains details about the cloud user.
  - **InEverybodyRole (boolean)**: Indicates if the user is in the Everybody role.
  - **Description (string)**: A description of the user.
  - **Name (string)**: The user's name.
  - **MobileNumber (string)**: The user's mobile phone number.
  - **LastPasswordChangeDate (string)**: The date and time when the user last changed their password.
  - **OfficeNumber (string)**: The user's office phone number.
  - **PictureUri (string)**: The file system path to the user's picture file.
  - **ReportsTo (string)**: The UUID of the user to whom this user reports.
  - **HomeNumber (string)**: The user's home phone number.
  - **DisplayName (string)**: The user's display name.
  - **PreferredCulture (string)**: The user's preferred culture.
  - **Alias (string)**: The login suffix of the user.
  - **State (string)**: The state of the user (e.g., locked, enabled, expired).
  - **PasswordNeverExpire (boolean)**: Indicates if the user's password never expires.
  - **Uuid (string)**: The UUID of the user.
  - **Mail (string)**: The user's email address.

- **Error (object)**: Contains error message text on failure, may be null.
