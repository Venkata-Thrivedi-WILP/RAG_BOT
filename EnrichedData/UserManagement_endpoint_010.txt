# Overview  
The **Get Users Details** endpoint allows you to retrieve a comprehensive list of all existing users in the cloud directory. This endpoint is accessed via a **POST** request at the path **/CDirectoryService/GetUsers**. It is specifically designed for use by system administrators and users with user management rights, ensuring that only authorized personnel can fetch user details. The API does not require any parameters in the body of the request. A successful response will return user IDs, names, and various details for all users present in the cloud directory. This endpoint can also be utilized to check if a user already exists before adding a new one. Access to this endpoint requires bearer authentication.
