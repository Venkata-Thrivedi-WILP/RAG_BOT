**Overview**
================

Get users details endpoint allows system administrators and users with user management rights to fetch the list of all existing users in the cloud directory. This API is a simple GET request which doesn't need any parameters in the body arguments.

### HTTP Method
POST

### Path
/CDirectoryService/GetUsers

### Security Requirements
Bearer authentication is required for this endpoint.

**Developer Notes**
==================

### Required Parameters
None

### Request Structure
The request body should be empty.

### Response Structure
```json
{
  "Result": {
    ...
  },
  "Error": {
    ...
  }
}
```

### Error Handling
If the user does not have administrative/user management permission, an error message will be returned with a code of "UnauthorizedUser".

### Security Considerations
Bearer authentication is required for this endpoint. The `x-idap-anon` metadata field indicates that anonymous access is not allowed.
