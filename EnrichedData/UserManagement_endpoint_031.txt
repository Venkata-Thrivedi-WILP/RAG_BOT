# Overview  
This API endpoint allows you to retrieve the risk level of a specified user from the Cloud Directory. It is accessed via a POST request at the path `/UserMgmt/GetUserRiskLevel`. Only system administrators with user management rights can use this API. To make a request, you need to provide the user's unique ID in the request body.

**Security Requirement:**  
This endpoint requires bearer token authentication.
