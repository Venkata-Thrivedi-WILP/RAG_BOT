**Overview**
===============

The `/AuthProfile/GetProfileMFAScoring` endpoint allows users to retrieve their authentication assurance level. This endpoint is used for security purposes and requires a valid bearer token for authentication.

To use this endpoint, send a POST request to the `/AuthProfile/GetProfileMFAScoring` path with the `challenges` query parameter containing profile factors.

**Developer Notes**
==================

### Required Parameters

* `challenges` (query, Required): Profile factors

### Request/Response Structure

The request body should contain the `challenges` query parameter. The response will be in JSON format and include a `Result` object containing the maximum and minimum AAL values, or an `Error` object with an error message if authentication fails.

### Error Handling

If authentication fails, the response will include an `Error` object with an error message. If successful, the response will contain a `Result` object with the maximum and minimum AAL values.

### Security Considerations

This endpoint requires bearer token authentication. The `x-idap-anon: False` metadata indicates that anonymous access is not allowed.
